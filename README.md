## 📖 Overview

A real-time American Sign Language (ASL) detection system using **TensorFlow**, **MediaPipe**, and **Streamlit**. This app recognizes hand signs from ASL fingerspelling (A-Z) and converts them to text and speech.

### 🌟 Features

- 🔍 **Image Upload** - Upload images of hand signs for detection
- 🎥 **Webcam Integration** - Real-time sign detection (desktop)
- 📝 **Sentence Building** - Combine detected letters into words/sentences
- 🔊 **Text-to-Speech** - Convert detected signs to spoken words
- 🎨 **Hand Landmark Visualization** - See how the model detects your hand
